Spacing direccional

Classes disponibles:
• .mt-* = margin-top
• .mr-* = margin-right
• .mb-* = margin-bottom
• .ml-* = margin-left
• .mx-* = margin horitzontal (left + right)
• .my-* = margin vertical (top + bottom)
(El mateix per padding: .pt-*, .pr-*, .pb-*, .pl-*, .px-*, .py-*)
mx-5 + my-3

mt-5
Marge superior de 40px

Spacing responsiu

Redimensiona la finestra del navegador per veure els canvis!

Les classes responsives segueixen aquest patró:
• .m-sm-* → a partir de 576px
• .m-md-* → a partir de 768px
• .m-lg-* → a partir de 992px
Marge inferior adaptable:
• Mòbil: 16px (mb-2)
• Tablet: 24px (mb-sm-3)
• Desktop S: 32px (mb-md-4)
• Desktop L: 40px (mb-lg-5)

Padding adaptable:
El padding interior canvia segons la mida de pantalla
